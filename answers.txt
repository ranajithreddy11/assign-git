1)What is a gitignore file and what should it include for a Java project?
Git ignore is a text while which we configure while creating a new repo. The primary use of it is, it tells  git to not track certain files.In java project, we ignore .class,.jar,.war,.ear,.nar etc.

2)What does fast-forward in git mean?
Fast-forward moves our master's branch's tip to the end of your feature branch. This keeps all commits created in your feature branch in sequence.
